# Portfólio One Page | Desenvolvedora Full Stack

Portfólio profissional one-page com design moderno, foco em performance e experiência do usuário.

## Descrição

Este projeto apresenta uma landing page de portfólio com seções de apresentação, habilidades, experiência, projetos, serviços, estatísticas e contato.  
Foi desenvolvido para ser responsivo, elegante e pronto para personalização.

## Tecnologias e Ferramentas

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (ES6+)
- CSS3 (Grid/Flexbox, animações e transições)
- FormSubmit (envio de formulário para e-mail)
- Favicon personalizado

## Funcionalidades

- Layout responsivo (mobile-first)
- Modo claro/escuro com persistência
- Navegação por âncoras com menu fixo
- Scroll suave e animações ao rolar
- Barras de skills animadas
- Contadores animados de estatísticas
- Formulário com validação e proteção anti-bot (captcha + honeypot)
- Botão de contato com WhatsApp e mensagem pré-definida

## Como Executar

1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.
